# Подсказки по командной строке

Команда смены директории
```sh
cd C:\folder_name
```

Команда отоброжения директории MacOS, Linux:
```sh
pwd
```

Список в текущей директории
Wondow:
```sh
dir
```
MacOS, Linux:
```sh
ls
``````

Удаление файла Windows
```sh
del <filename>
```
в MackOS, Linux:
```sh
rm <filename>
``````
