# Подсказки по Git

Создание репозитория
```sh
git init
```

Добавление файла для последующих  коментариев
```sh
git add
```
Добовление коментария 
```sh
git commit -m"Коментарий"
```
Просмотор списка изменений
```sh
git log
```
Просотр сокращенного списка изменений 
```sh
git log --oneline
```
